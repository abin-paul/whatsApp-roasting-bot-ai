# WhatsApp Roasting Bot

An AI bot using Llama 3 405b. It stores a massive context window of group chat history (up to 1000 messages) in a local SQLite database so it has perfect memory when deciding who and how to roast.

## Setup
1. Duplicate `.env.example` to `.env` and add your NVIDIA API Key.
2. Run `npm install`
3. Run `npm start`
4. Scan the QR code with your WhatsApp app.
